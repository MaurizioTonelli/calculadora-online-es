Esta es una calculadora sencilla construida con HTML, CSS y JavaScript.

El cálculo con jerarquía de operaciones se logra con gramáticas:

### Numero: 
- Número decimal

### Primario: 
- "("Expresión")" 
- Numero 
- -Numero 
- +Numero

### Término:
-Término * Primario
-Término / Primario

### Expresión:
-Expresión + Término
-Expresión - Término
